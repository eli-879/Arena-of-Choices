How To Play

Enter names of participants/choices into the text box on the right with one name/item per line.

When all items are entered, press start to begin simulation.

The order of deaths will appear on the left with the first death as 1, second death as 2, and so on.

If you are using this to select your teams: Use the death list in any way to split the teams, be it odds and evens, first X names, or however you like.

If you are using this to choose one item from many: Take the winner at the end.

Your data will from this run will be saved into a database where it can be viewed later.
